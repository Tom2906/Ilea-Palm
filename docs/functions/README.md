# Functions And Services

Document business logic and API/service responsibilities.

For each function/service, record:

- What it does
- Inputs/outputs
- Error behavior
- Side effects
- Callers/dependencies

Suggested sections:

- API Controllers (`EmployeeHub.Api/Controllers/*`)
- Services (`EmployeeHub.Api/Services/*`)
- Shared frontend logic/hooks (`employee-hub-ui/src/hooks/*`, `employee-hub-ui/src/lib/*`)
