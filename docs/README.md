# Ilea Palm Documentation

This folder is the long-term documentation source for developer handover.

## Structure

- `docs/operations/README.md`: Deployment and operations runbook
- `docs/features/README.md`: User-facing features and behavior
- `docs/functions/README.md`: Business logic/functions and service responsibilities
- `docs/components/README.md`: Frontend component inventory and usage
- `docs/handover/README.md`: Final handover checklist and environment notes

## Documentation Rules

- Keep docs updated in the same PR as code changes.
- Prefer concise, practical notes over long prose.
- Include file paths and API endpoints where relevant.
- Record assumptions and known limitations.
